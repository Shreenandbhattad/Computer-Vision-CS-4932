Part A

Harris Corner Detection

Precision-recall analysis using manually selected ground-truth corners.

Objectives

  1. Capture a building image with clearly visible structural corners.
  2. Manually annotate approximately 40 to 60 corners as ground truth.
  3. Run the Harris detector under multiple threshold values.
  4. Measure precision and recall at each threshold.
  5. Visualize detected corners and summarize the resulting trade-offs.
  6. Plot the precision-recall curve and estimate area under the curve.
