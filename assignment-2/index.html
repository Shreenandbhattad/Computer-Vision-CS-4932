Corner Detection and Feature Matching

Part A

Harris Corner Detection

  1. Capture a building image with visible corners.
  2. Mark about 40 to 60 corners as ground truth.
  3. Run the Harris detector with different thresholds.
  4. Measure precision and recall for each threshold.
  5. Visualize the detected corners.
  6. Plot the precision-recall curve.

Part A notebook was not included in the repository.

Part B

SIFT Feature Matching

  1. Extract SIFT keypoints and descriptors.
  2. Match features between consecutive image pairs.
  3. Calculate the total matching cost.
  4. Repeat the process for random pairs.
  5. Compare the results.
